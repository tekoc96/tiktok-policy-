Efsane Shorts Bot

Efsane Shorts Bot is a personal automation tool that creates short historical-education videos and uploads them to the owner's own TikTok account (@efsanelervegercekleryt). It is operated by a single individual for personal use and is not a commercial service offered to third parties.

Contact: tekinhardi@gmail.com

Last updated: June 2026

Privacy Policy

Data we access. Through the TikTok API, the app accesses only the owner's own TikTok account: basic profile information (open id, display name, avatar) and the ability to upload videos to that same account.

Data we store. The app stores a TikTok OAuth access/refresh token locally on the owner's own private computer, used solely to authenticate uploads. No data is stored on any public server.

Data we do NOT do. The app does not collect, process, sell, or share data from any other TikTok users. It does not run analytics on third parties, does not display ads, and does not transmit personal data to any third party other than TikTok's own API for the purpose of uploading the owner's content.

Data deletion. The locally stored token can be revoked at any time from the TikTok account settings or by deleting the local token file. No other personal data is retained.
